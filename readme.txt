Make your website feel interactive. Smart Text Interactions lets you customize text highlight colors and show a “Copied” tooltip when visitors copy content.

== Description ==

**Smart Text Interactions** enhances user engagement by:

- Letting you customize the background and text color for selected (highlighted) text
- Displaying a tooltip like “Copied” when a visitor copies any text
- Applying these features site-wide automatically
- Simple, fast, and lightweight — no shortcodes or setup required

Designed with simplicity and clarity in mind, this plugin works out of the box.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`  
2. Activate via the Plugins menu  
3. Visit **Settings → Smart Text Interactions** to customize your options

== Frequently Asked Questions ==

= Can I turn off the tooltip? =  
Not in this version. The tooltip is always on by default.

= Will this affect my site’s theme or styles? =  
No. It only applies highlight and tooltip styles site-wide.
